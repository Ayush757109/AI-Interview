# 🚀 AI Remote Interview Platform

A **Full Stack AI-powered Interview Platform** built with modern web technologies like **Next.js, Supabase, Tailwind CSS, Stream, Arcjet, and Shadcn UI**.
This platform allows users to practice interviews, get real-time feedback, and simulate real interview environments.

---

## 🌟 Features

* 🔐 **Authentication** (Clerk)
* 🤖 **AI-powered interview simulation**
* 🎥 **Real-time video & chat (Stream)**
* 📊 **Performance feedback & analytics**
* 🧠 **Smart question generation**
* ⚡ **Fast & scalable backend (Supabase)**
* 🎨 **Modern UI (Tailwind + Shadcn UI)**
* 🛡️ **Security & rate limiting (Arcjet)**

---

## 🛠️ Tech Stack

* **Frontend:** Next.js (App Router), React
* **Backend:** Supabase (Database & Auth)
* **Styling:** Tailwind CSS, Shadcn UI
* **Authentication:** Clerk
* **Real-time Communication:** Stream
* **Security:** Arcjet

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Interview.git
cd AI-Interview
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Setup environment variables

Create a `.env` file and add:

```env
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=

NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=

NEXT_PUBLIC_STREAM_API_KEY=
STREAM_SECRET_KEY=

ARCJET_KEY=
```

---

### 4️⃣ Run the project

```bash
npm run dev
```

👉 Open: `http://localhost:3000`
